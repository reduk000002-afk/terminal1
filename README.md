# GitHub Balance Editor

Простой интерфейс для редактирования `users.json` прямо в GitHub репозитории.

## Как использовать:

1. **Создайте Personal Access Token:**
   - Зайдите в GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
   - Создайте токен с правами **`repo`** (полный доступ к репозиториям)
   - Скопируйте токен (он покажется только один раз!)

2. **Настройте редактор:**
   - Откройте `index.html` (можно через GitHub Pages)
   - Введите ваш логин GitHub
   - Введите название репозитория где лежит `users.json`
   - Вставьте ваш токен
   - Нажмите "Сохранить настройки"

3. **Редактируйте:**
   - Изменяйте балансы в полях ввода
   - Нажимайте "Сохранить" для каждого пользователя или "Сохранить все изменения"

## Размещение на GitHub Pages:

1. В настройках репозитория (Settings → Pages)
2. Source: Deploy from a branch
3. Branch: `main` или `master`, folder: `/ (root)`
4. Сохраните

Через минуту ваш редактор будет доступен по адресу: `https://ваш-логин.github.io/balance-editor/`
